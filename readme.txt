IoTAgricultureMonitoringSystem
------------------------------
An IoT system that monitor Soil Moisture using WiFi with a automated irrigation system using relay switch.

Main Code
---------
The finalised and combined code (.ino) for the system.

PrototypeAndTestCode
--------------------
Codes (.ino) that was used for testing libraries or prototyping the functions. The main code functions are
tested here.

Info
----
Contain informations such as ThingsBoard Device ID and IP address of different ESP8266 board.
